1. Installation: 

`npm i`

2. Start Server: Run this on server where you need to transfer files

`npm start`

3. Know your system IP:
   Windows + R > type `cmd` > in command prompt type `ipconfig` > Check for IPv4 Address under Wireless LAN adapter Wf-Fi
   <img width="887" height="234" alt="image" src="https://github.com/user-attachments/assets/504b32cc-e250-4a38-8cc9-e623a8ba9ba8" />

4. Access application in devices from which you want to transfer the files.
   `192.168.xx.x:3000`  
5. Select Directory where you need to transfer file(Or create a new if required)
6. Click on Upload
